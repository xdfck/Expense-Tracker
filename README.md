# Random Quote Generator

**Автор:** Киселев Константин Максимович

## Описание

Приложение для генерации случайных цитат с графическим интерфейсом. Позволяет выбирать случайную цитату из предопределённого списка, фильтровать их по автору и теме, а также сохранять историю сгенерированных цитат в файл JSON.

## Технологии

- Python 3.x.
- PyQt5.
- random.
- json.
- Git.
